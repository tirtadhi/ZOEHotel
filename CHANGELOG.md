# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-27

### Added

- **Frontend Pages**

  - Homepage with hero section, features, featured rooms, testimonials
  - About page with company information and values
  - Rooms page with filtering, sorting, and search functionality
  - Room detail page with booking form and image gallery
  - Contact page with form and map integration
  - My Reservations page for user booking management
  - Sign In and Sign Up pages with validation
  - Admin Dashboard with statistics and analytics
  - Admin Room Management with CRUD operations

- **Components**

  - Navbar with responsive mobile menu
  - Footer with contact info and social links
  - Reusable Button component with variants
  - Input and TextArea components with validation
  - RoomCard component for room listings

- **Features**

  - Room filtering by category and price range
  - Search functionality across rooms
  - Sorting options (price, rating)
  - Responsive design for all screen sizes
  - Image gallery for room details
  - Booking form with date selection
  - Price calculation for multiple nights
  - Reservation status tracking
  - Admin statistics dashboard
  - Room availability management

- **Data Layer**

  - TypeScript interfaces for type safety
  - Mock data for rooms and bookings
  - Helper functions for formatting
  - Constants for configuration
  - Utility functions for common operations

- **SEO & Performance**

  - Meta tags and Open Graph images
  - Sitemap generation
  - Robots.txt configuration
  - Optimized images with Next.js Image component

- **Documentation**
  - Comprehensive README
  - Deployment guide
  - Contributing guidelines
  - Environment variables example

### Technical Details

- Built with Next.js 16 (App Router)
- TypeScript for type safety
- Tailwind CSS 4 for styling
- React Icons for icon library
- Responsive and mobile-first design
- SEO optimized
- Performance optimized
